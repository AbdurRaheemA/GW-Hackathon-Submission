## GW Hackathon Submission
#### Track 1
#### Problem 3
##### Group Members: Ashley Totten, Amelia Tran, Abdur Raheem Ali, Mikael Makonnen

#### Question
AI For Nutritional Awareness
Create a solution that turns complex nutrition data into simple, clear guidance people can easily follow.

#### Our Solution
Personalized Dietary Planner which incorporates elements of calorie, and nutritional balance.
- Inputs such as height, weight, gender, and dietary restrictions

- Meals for the day (generic)

#### Deliverables
1. Personalized AI Model which takes user health data and produces personalized daily calorie/nutrition intake
2. Utilizing AI model outputs to filter through recipe API to produce desired recipes that fit nutritional guidelines

#### Extras
- Cuisine Choice
- User Interface
